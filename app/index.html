Node.js installatie

Node.js is een javascript-implementatie om javascript zonder browser te kunnen draaien.

Om de Node.js interpreter werkend te krijgen zijn er bepaalde stappen nodig die uitgevoerd moet worden.

Ga naar de opgegeven link klik hier en selecteer de versie open source.

Git clone

Als eerst moet u de server clonen om vervolgens de benodige data te kopiëren.

$ git clone
